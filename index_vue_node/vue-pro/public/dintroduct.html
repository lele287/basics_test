医生介绍

医学教育背景介绍 : 中国人民解放军第四军医大学西京医院副主任医师；1983年 毕业于中国人民解放军第四军医大学。
擅长疾病 : 湿疹、皮炎、荨麻疹、银屑病等。疑难皮肤病：重症痤疮。结缔组织病：硬皮病、皮肌炎、红斑狼疮。血管炎：过敏性紫癜、结节性红斑。面部疑难问题：激素依赖性皮炎、痤疮炎症后色素沉着及瘢痕、黄褐斑、皮肤黑变病。 注：年已花甲，不再坐诊，若需服务，请来春雨。
医院地点 : 空军军医大学西京医院 陕西省西安市新城区长乐西路15号【春雨提示：部分医院有多个院区，请先与医生确认好地点后再前往就诊】
